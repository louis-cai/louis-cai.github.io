马丁策略–如何使用

发表于 2021-05-19 更新于 2021-09-07 分类于 交易策略 阅读次数：
原文地址: Martingale Strategy – How To Use It

如果您有一段时间从事外汇交易，就有可能听说过马丁格尔。但是它是什么以及它如何工作？在这篇文章中，我将讨论该策略，它的优势，风险以及在现实世界中如何最好地使用它。

这种策略对货币交易者具有吸引力的原因有几个。

首先，在一定条件下，它可以带来可预测的利润结果。这不是一个确定的赌注，但它尽可能地接近您。

其次，它不依赖于预测绝对市场方向的能力。考虑到外汇的动态和波动性，这很有用。

拥有足够多的资金，当您的交易挑选技巧比偶然更好时，它就可以工作。尽管它的确取得了更好的结果，并且损失更少，但是您对未来市场的预测越熟练。

第三，货币往往会在很长一段时间内进行区间交易-因此，多次都反复讨论相同的水平。与网格交易一样，这种行为也适合这种策略。

马丁格尔是一项平均成本策略。它通过在亏损交易中“使风险敞口加倍”来做到这一点。这样会降低您的平均入门价格。

要了解马丁策略，重要的是，这不会增加您获胜的几率。您的长期预期收益仍然完全相同。这取决于您能否成功选择成功的交易和正确的市场。您无法逃脱。

该策略的作用是延迟损失。在适当的条件下，损失可能会延迟这么多，以至于看来是肯定的事情。

马丁策略怎么运作

简而言之：马丁策略是一项平均成本策略。它通过在亏损交易中“使风险敞口加倍”来做到这一点。这样会降低您的平均入门价格。这个想法是，您只是继续将交易规模扩大一倍，直到最终命运使您单笔获胜交易成功。到那时，由于倍增效应，您可以获利退出。

一个简单的输赢游戏

这个简单的例子说明了这个基本思想。想象一下一个交易游戏，有50:50的机会赢得失败的诗歌。

我下注了1美元的交易。每次获胜时，我的赌注保持不变，为$ 1。如果我输了，我每次将赌注翻倍。赌徒称之为加倍下注。

如果赔率合理，那么最终结果将对我有利。而且由于我每次都将赌注翻倍，因此当这种情况发生时，获胜者将恢复之前的所有损失以及原始赌注。

这要归功于双重效应。下注总是赢利。由于$2^n=\sum2^{n+1}+1$的数学事实，这是正确的。这意味着一连串的损失可以通过最后一次获胜的交易来弥补。

基本交易系统

在真实交易中，没有严格的二元结果。一笔交易可以带来一定的盈亏。但这并没有改变基本策略。您只需将基础价格的固定变动定义为止盈和止损水平。

以下情况说明了这一点。我将止盈和止损设在20点。

我首先以1.3500的价格开1手买单。然后，汇率对我不利，跌至1.3480，下跌了20点。它达到了我的虚拟止损。

这实际上是一个止损，因为在实际交易中，平仓并以两倍大的规模开立新头寸是没有意义的。我将现有的每条腿保持开放状态，并添加新的贸易订单以将其大小增加一倍。

因此，在1.3480，我通过增加1手来使我的交易量增加一倍。这使我的平均持仓成本降为1.3490。我的损失是一样的，但是现在我只需要+10点的回撤即可达到收支平衡，而不是以前的20点。

“平均下降”的行为意味着您的交易量增加了一倍。但是，您还可以减少弥补损失所需的相对金额。表2中的“收支平衡”列显示了这一点。

当您平均交易更多交易时，盈亏平衡点接近恒定值。这个常数值越来越接近您的止损。这意味着即使只有很小的回撤，您也可以迅速抓住“下跌的市场”并弥补损失。

无论市场相对该头寸移动了多远，标准马丁策略鱼都会始终在一个止损距离内恢复。（请参见图1）。

在第5次交易中，我的平均持仓成本现在为1.3439。当汇聚之后上升到1.3439时，它达到了我的收支平衡。

一旦汇率达到或高于盈亏平衡水平，我就可以关闭交易系统。我的前四笔交易止损关闭。但都完全被序列中最后一笔交易的利润所覆盖。

马丁策略总是奏效吗？

在纯粹的马丁策略系统中，没有完整的交易序列会亏损。如果价格对您不利，您只需将交易规模扩大一倍。

但是，这样的系统在现实世界中是不存在的，因为它意味着拥有无限的货币供应和无限的时间。两者都不是可以实现的。

在真实的交易系统中，您需要为整个系统的亏损设置限制。一旦您通过了止损线，交易顺序就会亏损。然后该循环再次开始。

当您限制盈利能力时，您将不再使用纯马丁策略系统。这样做时，您将一直使用一个近似值，该近似值将始终存在一个无效点。

加倍的经文损失的概率

窘境是，您的止损限额越大，亏损的可能性就越低，但亏损会越大。这就是塔莱布的困境。

您进行的交易越多，这些极端的可能性就越有可能“浮现”，而一连串的损失将使您破产。

在马丁策略中，亏损序列上的贸易敞口呈指数增长。这意味着在N笔亏损交易的序列中，您的风险敞口将增加为$2^{n-1}$。因此，如果您被迫过早退出，损失可能真是灾难性的。

另一方面，获胜交易的利润仅线性增加。它与每笔交易利润的一半乘以交易总数成正比。

获胜的交易总是会在这种策略中创造利润。因此，如果您在50％的时间内（至少没有机会）选择获胜者，则获胜交易的总预期收益为：
$$E≈\frac{1}{2}N \times B$$

其中N是“交易”的数量，B是每笔交易的获利金额。

但是，您一笔大的亏损交易会将其设置为零。例如，如果您的限制是10次双倍加仓交易，则最大交易为1024。只有连续11个亏损交易，您才会亏损此金额。这样的可能性是$({\frac{1}{2}})^{11}$。这意味着，每2048个交易，您可能会亏损一次。

因此，在2048次交易之后：

  • 您的预期奖金是$\frac{1}{2}\times2^{11}\times1=1024$
  • 您的预期一次性亏损是-1024
  • 您的净利润为0
    因此，在真实系统中，您的赔率始终保持在50:50。前提是您的交易选择总比机会好。

您的风险回报也与1：1保持平衡。但是，与大多数其他策略不同，在马丁策略中，您的损失很少但非常大。因此，管理起来可能很困难，尤其是如果您很不幸，而且这种情况发生在您有机会积累任何利润之前！

所有这一切的要点是，马丁无法提高您获胜的几率。它只是推迟您的损失。参见表4。

那些本质上是趋势追随者的人通常认为，最好使用反向马丁策略。反马丁策略或反向马丁策略试图做与上述完全相反的事情。基本上，这是一种遵循趋势的策略，可以使获胜者加倍，并迅速减少损失。

远离“趋势”货币

根据我的经验，该策略的最佳机会来自范围交易。并且通过将交易规模与资金成比例地保持很小就可以使用非常低的杠杆。这样，您就有更大的承受力，可以承受缩编中出现的更高交易倍数。

根据我的经验，马丁策略是最有效的增产剂。

当然，还有许多其他观点。有人建议使用马丁策略结合套利交易。这意味着利率差很大的交易对。例如，使用AUD/JPY的多头交易策略。

这个想法是由于大量的未平仓交易量积累了正的展期信用额度。

但是，这种方法存在问题。风险在于，带有套利机会的货币对通常会遵循强势趋势。这些工具通常会在解开携带位置（逆向放置位置）时看到陡峭的校正期。

这可能会突然发生，而不会发出警告。例如，如果利率周期发生意外变化，或者风险偏好突然发生变化，在这种情况下，资金往往会很快从高收益货币中撤离（更多关于套利交易的信息。）

分析表明，从长远来看，马丁策略在趋势市场上的表现非常差（请参阅收益表）。

还有一点要牢记，许多经纪人将套利交易带给了很大的价差-这使得除了收益率最高的套利交易之外的所有交易都无利可图。一些零售经纪人甚至根本不相信积极的展期。

最后，低收益率意味着您的交易规模需要与资本成比例，以使套息能够对结果产生任何影响。如上面的示例所示，这对马丁策略来说太冒险了。

相反，更适合趋势的策略是反向马丁策略。

使用马丁策略作为收益增值产品

马丁策略不应作为主交易策略。这是因为要使其正常工作，您需要相对于您的交易规模具有较大的亏损限制。如果您使用大量的交易资金，那么在下降的趋势之一中，确实存在“破产”的真正风险。

以我的经验，更好地使用马丁策略是作为低杠杆的收益增值产品。

风险最小的交易机会是在小范围内交易的货币对。

波动率工具可用于检查当前市场状况以及趋势。最好的配对是那些倾向于在策略中蓬勃发展的长期绑定期的配对。

马丁策略可以在趋势中生存，但前提是必须有足够的回调。这就是为什么您必须提防重大新趋势的突破–特别要注意关键支撑/阻力位。

具有强趋势行为的交易对（例如日元交叉货币或大宗商品货币）可能具有很大的风险。

计算您的止损限额

一个很好的起点是确定您可以冒险的最大未平仓手数。由此，您可以计算出其他参数。为简单起见，我将使用2的幂。

最大手数将设置在平仓之前可以通过的止损水平数量。换句话说，这是该策略“加倍执行”的次数。因此，如果您的最大总持仓量为256手，这将允许翻倍8倍-或8脚。关系是：

$${max\ lots = 2^{Legs}}$$

如果您在第n个止损位平仓整个头寸，您的最大损失将是：

$${max\ loss = (2^n-1)\times s}$$

s是止损距离（以点为单位），在该距离处您将头寸规模扩大一倍。因此，如果有256手（微型手），止损为40点，那么在第8个止损水平收盘时，最大亏损为10,200点。在第9个止损位收盘将亏损20,440点。

提示 计算出亏损前您可以处理的平均交易数–使用公式$2^{Legs+1}$。因此，在这里的示例中，只有$2^9$笔交易，即512笔交易。因此，在进行512笔交易后，您将有9位输家获得偶数赔率。这会破坏您的系统。

您可以使用Excel工作簿中的手数计算器来尝试不同的交易大小和设置。

决定进入信号

仍然需要在某些时候触发该系统如何开始买卖。可以使用任何有效的买入/卖出信号，但信号越好，该策略将越有效，跌幅就越小。

在这里的示例中，我使用的是简单的移动平均线。当汇率移动到移动平均线上方一定距离时，我发出了卖单。当它移动到移动平均线以下时，我下了一个买单。该系统正在交易错误的突破，也称为“衰落”。

在我的系统中，我使用15点移动平均线（MA）作为输入信号。您选择的移动平均线长度将取决于您的特定交易时间范围和一般市场条件。

这是一个非常简单且易于实现的触发系统。您可以尝试更多更复杂的方法。例如，使用布林通道，其他移动平均线或任何技术指标的差异。

强大的突破动作可能导致系统达到最大损失水平。因此，在波动紧缩时以及在数据发布之前，应尽量靠近关键支撑/阻力区域进行交易。

设置止盈和止损

接下来要考虑的两点是

什么时候加倍–这就是您的虚拟止损
何时关闭–您的“止盈水平”
何时加倍–这是系统中的关键参数。“虚拟”止损意味着您假设此时交易不利于您。这是一个失败者。因此，您的交易量翻了一番。

选择的价值太小，您将开设过多的交易。太大的价值会阻碍整个战略。

您为止损和获利选择的价值最终应取决于您交易的时间范围和波动性。较低的波动率通常意味着您可以使用较小的止损。我发现20到70点之间的值在大多数情况下都很好。

何时关闭ting交易只应在“整个系统”获利时关闭。也就是说，当未平仓交易的净利润至少为正时。与网格交易一样，在马丁策略中，您需要保持一致，并将交易集作为一个整体而不是独立地对待。

较小的止盈值（通常为10-50点）通常在此设置中效果最好。

有两个原因。

止盈水平越小，越早获得利润的可能性就越高，因此您可以在系统获利的时候关闭。
由于交易的手数成倍增加，所以利润变得更加复杂。因此，较小的值仍然有效。
使用较小的获利不会改变您的风险回报。尽管收益较低，但越接近胜利阈值，您的总体贸易胜利率就越高。

马丁策略的利与弊

为什么使用它：

  • 它具有一组定义明确的交易规则，可以很容易地遵循这些规则或将其编程为EA交易。
  • 就利润和亏损而言，它具有统计上可计算的结果。
  • 如果应用正确，它可以实现递增的利润流。
  • 您不需要能够预测市场方向。

为什么要避免：

  • 平均下降是一种避免损失而不是寻求利润的策略。
  • 马丁策略不会增加您获胜的几率。如果您很幸运的话，它只会延迟损失很长时间。
  • 它依赖于关于随机市场行为的假设，这些假设并不总是有效的。市场的行为确实不合理。
  • 风险敞口呈指数增长，而利润呈线性增长。
  • 在实践中，由于没人拥有无限量的资金，它有可能带来灾难性的损失。
  • 风险与报酬是平衡的，但是由于损失很大，因此可能无法接受。
